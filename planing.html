Estructura básica de HTML5

Una estructura HTML se empieza con la etiqueta <"html"> y acaba con <"/html">. Todo lo que esté en medio será la página web. Dentro de <"html"><"/html"> se encuentran 2 partes diferenciadas.La primera <"head"><"/head"> es la cabecera de la página. Aquí irán cierta información que no es directamente el contenido de la página. Aquí se pone el título de la página, los metadatos, estilos, código javascript.

La segunda parte es <"body"><"/body">. Aquí va propiamente el contenido de la página: fotos, párrafos, formularios, etc. Ver más

Principales etiquetas de HTML5

Estas son algunas etiquetas en HTML5:

No. ETIQUITA HTML DESCRIPCIÓN
1 HTML Indica el comienzo y el final del documento html.
2 HEAD En esta se ubica el encabezado del documento.
3 BODY Es el cuerpo del documento.
4 TABLE Se usa para indicar tablas.
5 DIV Se usa para indicar una división dentro del contenido.
6 BR Se usa para indicar un salto de línea en el texto.
7 H1 Se usa para crear títulos en el documento.
8 P Se usa para añadir una línea o un párrafo de texto.
9 STRONG Esta etiqueta le da énfasis al texto que contiene.
10 IMG Se usa para añadir imágenes.

Atributos y valores de HTML5

Los atributos básicos que se utilizan en la mayoría de etiquetas HTML son los siguientes:

No. ATRIBUTO HTML DESCRIPCIÓN
1 id="texto" Establece un indicador único a cada elemento.
2 class="texto" Establece la clase CSS que se aplica a los estilos del elemento.
3 style="texto" Aplica de forma directa los estilos CSS de un elemento.
4 title="texto" Establece el título del elemento (Mejora la accesibilidad).
